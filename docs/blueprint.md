# **App Name**: Lunar Attraction

## Core Features:

- Hero Section Animation: Animated orbs of light/blurred backgrounds in blue and purple that create a captivating first impression.
- Scroll-Triggered Reveal: Framer Motion is used to fade in sections of content on scroll, improving engagement and creating a dynamic user experience.
- Energy Visualizer: Gamified visualizer that compares 'before' (low energy/anxiety) and 'after' (high energy/attraction) states.
- Dynamic Content Loading: As the user scrolls, additional sections of the landing page are loaded dynamically with subtle animations to keep the experience smooth and engaging. It analyzes the type of device that accesses the webpage (mobile/desktop) to generate content suitable to each one.

## Style Guidelines:

- Primary color: Vibrant cyan (#00FFFF) to evoke a futuristic and energetic feel.
- Background color: Dark slate gray (#2F4F4F), almost black, provides a premium, mysterious dark mode environment with glows.
- Accent color: Deep violet (#8A2BE2) to highlight key interactive elements.
- Font pairing: 'Poppins' for headlines (geometric sans-serif) and 'Inter' for body text (grotesque-style sans-serif).
- Use Lucide React icons throughout the design to ensure consistency and clarity; incorporate smooth transitions.
- Implement a glassmorphism effect on cards and sections, giving the UI a modern, frosted glass appearance.
- Incorporate Framer Motion to bring elements to life with subtle, fluid animations. Includes smooth fade-ins, floating elements, and gentle light flickers.
